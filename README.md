# Tutorial-M2
tutorial membuat ayay
## 1. install nvm (dilaptop)
- **curl -fsSL https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash**
- **source ~/.bashrc**
## 2. install Node 22
- **nvm install 22**
- **nvm use 22**
- **nvm alias default 22**
## 3. Cek Versi nvm dan Node
- **node -v**
- **npm -v**
## 4. Install aws configure dan sambungkan
## 5. Buat 2 Bucket
- **Buat 2 bucket yaitu untuk frontend dan Upload Gambar**
### A. Bucket untuk Frontend
- **Bucket name: nama-bucket**
- **Region: Sesuai Soal**
- **Bloclk: all public access: OFF**
- **Ke Tab Properties-> Static website hosting(Enable)-> index html semua**
### B. Bucket Untuk Upload
- **Bucket Name: sesuai soal**
- **Region: Sesuai Soal**
- **Blok all public access: ON**
